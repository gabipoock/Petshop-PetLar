## 📜 Formulário
O formulário possui campos para o preenchimento do cadastro. <br>
Devem ser completados: <br>
-Nome;<br>
-Email;<br>
-CPF;<br>
-Celular;<br>
-Data de nascimento;<br>
-Sexo;<br>
-Opção de ofertas.
